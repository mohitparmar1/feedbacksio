# feedbacksio
Description: send anonymous feedback to user 

## Tech Stack :
 - Nextjs
 - Tailwind Css
 - Next-Auth
 - MongoDB 

## features :
 1. Login
 2. Sign-up
 3. Dashboard
 4. Send Messages
